Supervised Machine Learning / NLP 2025

Email Spam Classifier

A production-grade email spam detection system built using TF-IDF vectorization and Multinomial Naive Bayes, trained on 5,169 labelled emails. Class imbalance was addressed using SMOTE, and the final model was deployed as a real-time Streamlit web app where users paste email text and receive an instant spam/not-spam prediction.

Key Outcomes

1

Achieved 97.6% overall accuracy and 100% precision on the spam class

2

Applied SMOTE to handle dataset imbalance, improving model fairness and recall

3

Trained and evaluated on 5,169 labelled emails across spam and ham categories

4

Delivered a live, user-facing prediction interface via Streamlit deployment

Technologies:
Python Machine Learning NLP TF-IDF Multinomial Naive Bayes Scikit-learn SMOTE Streamlit
